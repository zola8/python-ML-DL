# When and what to use


## Line Chart

When to use:

- To visualize **continuous data** and track trends or changes over time (Time-Series Data).
- Highlighting Patterns
- Comparing Multiple Groups: Plotting 2 to 4 different datasets on the same axis to compare their relative progression over the same period.
- Forecasting: Helping viewers infer directionality and predict what might happen next based on historical data.

When to avoid:

- Discrete Comparisons: If you want to compare distinct categories (use a Bar Chart instead).
- Short Time Spans: If you only have 4 or 5 data points, a column chart is usually better because the overall shape won't reveal much.


## Bar Chart

Bar chart displays **categorical data** using rectangular bars whose lengths are proportional to the values they represent. 
It can be plotted vertically or horizontally to compare different categories.

When to use:
- "Which is bigger?"
- Ranking
- Frequency distribution (Histogram)


## Histogram

Histogram shows the distribution of data by grouping values into bins.


## Scatter Plot

Use a scatter plot when you need to analyze the relationship between two continuous numeric variables. 
They are the ideal tool for spotting correlations, identifying clusters, and detecting outliers at a glance.

- Identifying Correlations
- Detecting Outliers
- Segmenting Data: To group data points together

When to Avoid a Scatter Plot:
- Categorical Data
- Time-Series Data
- Overcrowded Data


## Pie Chart

Use a pie chart exclusively to show part-to-whole relationships where all segments add up to 100%.

When to Use a Pie Chart:
- Few Categories
- Skewed Data: They work best when there are clear, obvious differences in the sizes of your categories (e.g., one slice is 75% and the others are small)

When to Avoid Them:
- Precise Comparisons
- Negative Numbers


## Box Plot

Box plot is a simple graph that shows how data is spread out. 

It displays the minimum, maximum, median and quartiles and also helps to spot outliers easily.
